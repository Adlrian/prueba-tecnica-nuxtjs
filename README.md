# prueba-tecnica-nuxtjs
Ejemplo lanzado en:

## Instrucciones
```bash
# clonar el repositorio
$ git clone 

# instalar dependencias
$ npm install

# compilar con hot reaload en localhost:3000
$ npm run dev

# compilar para produccion y lanzar
$ npm run build
$ npm run start

# generar proyecto estatico
$ npm run generate
```
